== Description ==

Simple Backup Plugin by Siteguarding is the most fast and simple WordPress backup plugin. We offer the easiest way to backup, restore and migrate your website. You can backup and restore your CMS files, database or both on your computer or on our secure servers.

<h4>Backup features</h4>
<ul>
<li>Database Backup</li>
<li>Core Files Backup</li>
<li>Migration to a third party servers</li>
<li>Choose which database to backup</li>
<li>Restore site in one click</li>
<li>Selective files or database restoration</li>
<li>Encrypt backup archives and save them on our hosting and servers.</li>
<li>Store backup to FTP server</li>
<li>Store backup to directory</li>
</ul>

<h4>MIGRATE WORDPRESS TO OUR CLOUD STORAGE SERVICES USING OUR FAST AND SIMPLE EXTENSION</h4>

We have a comprehensive Quality Assurance and testing process that ensures that the plugin is always compatible with the latest release of WordPress.

<h4>The plugin does not have any dependencies, making it compatible with all PHP hosting providers. There is an option to apply an unlimited number of find and replace operations on your database during the export process. The plugin will also fix any serialisation problems that occur during the find/replace operation.</h4>

Simple Backup helps you to migrate in all of these cases in the smoothest possible way. You just have to Backup your site and Restore it in the new location. No additional work will be required, since we handle all changes for you.

Simple Backup will help you to skip all possible problems with backup and migration, because of its advanced refactoring and migrating engine.

== Frequently Asked Questions ==

= Can I use Simple Backup to migrate a website? =

The answer is Yes, you can! Simple Backup supports advanced migration functionality which will help you easily migrate your website from one domain to another. All urls, images and other domain-dependent data will be properly refactored and deployed for the new domain.

= Can I use Your Servers To Save a Backup? =

You sure can. You can see the options in plugins control panel.

= How do I Restore my Backup? =

You just have to upload a zip archive to a new server and run the restoration process.

= Something is not working, what should I do? =

No software is perfect and we are not the exception. There are tons of different servers and configurations and something may go wrong in one of them.

Just contact us and we will be more than happy to help you.
